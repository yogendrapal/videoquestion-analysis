# Grpc python stream client

## Installation

### Ubuntu
- apt-get install python3.6
- apt-get install python3-pip

### Mac OSX
- `brew install python3`
- `brew install python3-pip`

___
- `pip3 install -r requirements.txt`
- update details in `user.config`
- `python3 client.py`
